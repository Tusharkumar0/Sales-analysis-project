# 📊 Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Sales Dashboard** developed in **Microsoft Power BI** to analyze business performance and provide valuable insights into sales trends, revenue, profit, and customer behavior.

The dataset was first cleaned and prepared using **Microsoft Excel**, then imported into **Power BI** for data modeling, visualization, and dashboard creation.


## 🎯 Objectives

- Analyze overall sales performance
- Track profit and revenue trends
- Identify top-performing products and categories
- Monitor regional sales performance
- Provide interactive filtering for better decision-making


## 🛠 Tools & Technologies

- 📊 Microsoft Power BI
- 📈 Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)


## 📂 Dataset

The dataset contains sales-related information including:

- Order Details
- Customer Information
- Product Category
- Sales
- Profit
- Quantity
- Region
- Order Date


## 🔄 Data Preparation (Excel)

Before importing the data into Power BI, the dataset was cleaned using Microsoft Excel.

### Cleaning Steps

- Removed duplicate records
- Handled missing values
- Corrected data formats
- Standardized column names
- Verified data consistency
- Prepared the dataset for analysis


## ⚙ Power BI Development

The dashboard includes:

- Data Modeling
- Power Query Transformations
- DAX Measures
- Interactive Slicers
- KPI Cards
- Charts and Graphs


## 📈 Dashboard Features

- 📌 Total Sales
- 💰 Total Profit
- 📦 Total Quantity Sold
- 🌍 Regional Sales Analysis
- 📅 Monthly Sales Trend
- 🛍 Category-wise Sales
- 🏆 Top Performing Products
- 🎛 Interactive Filters and Slicers


## 📊 Key Insights

- Identified the highest-performing sales regions.
- Compared monthly sales and profit trends.
- Evaluated category-wise business performance.
- Analyzed customer purchasing patterns.
- Tracked business KPIs for better decision-making.
